# ✨ AI Humanizer Studio

An advanced, open-source AI text humanization web app designed to completely bypass AI detectors (like ZeroGPT, Turnitin, etc.) by mathematically destroying LLM structural fingerprints.

## 🚀 How It Works

AI detectors look for specific structural tokens (like bullet points, formal transitions, and predictable sentence lengths). This pipeline physically shreds those structures using a **5-Step Gauntlet**:

1. **The "Tired Student" Persona (LLM Pass):** The input text is fed to an LLM (configured in `config.toml`) with strict instructions to dissolve all bullet points, vary sentence lengths randomly, and write like a tired university student at 2 AM before a deadline.
2. **Japanese Scramble (SOV):** The text is translated to Japanese (Subject-Object-Verb syntax) and back to English, forcing the verbs to the end of the sentence and scrambling syntax.
3. **Tamil Scramble:** The text is translated to Tamil and back, applying Dravidian language structural rules to further scramble nouns and adjectives.
4. **Russian Scramble:** The text is translated to Russian (a highly inflected language) and back, completely destroying any remaining morphological structures.
5. **Python Vocabulary Wipe:** Finally, a raw Python script runs a Regex sweep to physically delete all remaining "AI words" (e.g., *delve*, *leverage*, *crucial*, *robust*).

By the time the text reaches the end of the gauntlet, the final English output is technically generated by Google's Statistical Machine Translation engine, NOT an LLM. This ensures a 0% AI detection score while preserving the core meaning of the text.

## 🛠️ Installation & Setup

1. Clone this repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your configuration file:
   ```bash
   cp config/config.example.toml config/config.toml
   ```
4. Open `config.toml` and add your LLM API key (e.g., DeepSeek, OpenAI, or OpenRouter).

## 💻 Running the App

Start the FastAPI server:

```bash
python server.py
```

Then, open your web browser and navigate to:
**http://127.0.0.1:8000**

Paste your AI-generated text into the UI, click **Humanize Text**, and watch the pipeline break down the text step-by-step.

## 📁 Repository Structure
- `server.py` - The FastAPI backend server
- `static/index.html` - The frontend web UI
- `config/` - Configuration files for API keys
- `src/standard/` - The core humanization pipeline modules (`pipeline.py`, `llm_rewriter.py`, `translators.py`, `vocab_swapper.py`)
